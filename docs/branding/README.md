# Vodr Icon Assets

- `play-store-icon.png`: square `512x512` export for Play Store listing use.
- Android launcher icons ship from vector resources under `app/src/main/res/`.
- Adaptive icon entry points live at `app/src/main/res/mipmap-anydpi-v26/`.
- Pre-26 fallback icons live at `app/src/main/res/mipmap-anydpi/`.
- Themed launcher support uses `app/src/main/res/drawable/ic_launcher_monochrome.xml`.
